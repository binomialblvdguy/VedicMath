## What the App Does

VedicMath focuses on a few core learning flows:

- **Ratio / Multiply**
    - auto-observation of multiplication patterns
    - near-base multiplication
    - vertical and crosswise
    - same-units patterns
    - reciprocal-style patterns
    - grouping-based multiplication paths

- **Square**
    - auto-selected square methods
    - duplex-style and ending-based shortcuts

- **Cube**
    - auto-selected cube methods

- **Cross Product Quiz**
    - standard quiz flow
    - advanced quiz flow
    - scoring and session-based question progression
